Meditating

Meditating is awesome. Everyone says it, nobody does it, it's always something that you know you should do. Well, you have a couple minutes with nothing to do. Let's give it a shot.

It has a huge range of health benefits. The kicker for me, however, was thinking of it as exercise for my mind. If we're all so concerned with running and swimming and exercising my muscles and heart and all that, then shouldn't I do the same thing for my mind?

It's simple, it's free, it's immediate. Let's try it out, and see if it treats us better than Facebook.

If you've never meditated before, it can be hard to start. I've provided a very basic meditation "app" here with some basic prompts, but there are more resources at the bottom of the page.

If you have the HeadSpace app, just click here.
